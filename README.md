## 목적
- 개발 주제
  * 영어단어장 만들기 - 단어를 외우는 것을 도움.
  
- 구현 환경
  * Anroid 8.0

- 개발 언어
  * JAVA, PHP, XML
 
 - 팀 구성
  * 이동석, 정창현, 김태웅, 김선재, 한정용
